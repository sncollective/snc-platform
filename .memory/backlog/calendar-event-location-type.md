# Calendar event location type — physical, virtual, hybrid

Events need a structured location type field: Physical, Virtual, or Both (hybrid). Shows in particular need richer location properties depending on type — for example, a virtual show would carry a streaming or meeting link, while a physical show would carry a venue address, and a hybrid show would carry both. Ticketing links are an additional property that should be available for physical and hybrid events.

Note: event visibility and public event endpoints shipped in release-0.2.7. This item covers the location type enumeration and the per-type property set, which remain unbuilt.
