# ActivityPub Go-Live Notifications via Fedify

Send ActivityPub go-live notifications to followers when a creator starts a live stream. This was previously free with Owncast; with the unified streaming server decision (platform-built SRS), it must be implemented directly via Fedify dispatchers rather than delegated to a third-party service.
