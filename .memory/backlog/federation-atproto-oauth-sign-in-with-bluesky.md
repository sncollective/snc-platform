# AT Protocol OAuth — Sign In with Bluesky

Implement AT Protocol OAuth 2.1 with DPoP, PKCE, and PAR for "Sign in with Bluesky" support. This allows users with Bluesky or compatible AT Protocol accounts to authenticate on S/NC using their existing identity.
