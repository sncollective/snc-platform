# AT Protocol PDS Repo Mirroring with Custom Lexicons

Mirror S/NC content to the self-hosted AT Protocol PDS repository so it auto-distributes via the relay network. Define custom lexicons (`com.snc.media.*`) to represent S/NC-specific media content types within the AT Protocol data model.
