# NIP-05 Nostr Identity Verification

Serve `/.well-known/nostr.json` so that `creator@s-nc.org` resolves to a creator's Nostr pubkey. NIP-05 verification lets creators present a human-readable Nostr identity anchored to the S/NC domain.
