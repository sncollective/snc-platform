# Nostr Adapter

Implement the Nostr protocol adapter using `nostr-tools` (v2.23+) for event creation and signing, NDK for relay management, and `hono-nostr-auth` for NIP-98 HTTP authentication. This adapter covers Nostr event publishing and relay interaction.
