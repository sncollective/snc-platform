# Creator Recordings Management Page

A management page for creators to view, rename, and delete their past VOD recordings. Lists recordings from the creator's stream history with status (processing, ready, failed), duration, and file size. Scoped to Phase 7 VOD Pipeline — depends on SRS DVR recording and the recording post-processing pipeline producing stored MP4 files in Garage S3.
