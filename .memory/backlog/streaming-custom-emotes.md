# Custom Emotes

Platform-wide and creator-specific custom emotes in live chat. Platform-wide emotes are available to all; creator-specific emotes are uploaded by creators and scoped to their channel. Requires emote upload/management UI, storage in Garage S3, and emote parsing in the chat message renderer.
