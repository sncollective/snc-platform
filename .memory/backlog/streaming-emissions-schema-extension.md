# Emissions Schema Extension for Streaming

Extend the emissions tracking schema with `streaming-transcoding` and `streaming-delivery` categories. Streaming-transcoding covers CPU/energy cost of SRS + Liquidsoap live transcoding; streaming-delivery covers CDN/bandwidth for HLS delivery to viewers. Part of Phase 8 scope — hooks into the broader emissions calculation engine.
