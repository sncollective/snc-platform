# Playout Channel Architecture Rethink: Phase 3 Admin UI

Channel tabs in the playout admin, per-channel queue view and content pool management, content search picker spanning both playout items and creator content, and schema migration for `contentId` on queue entries. Design brief at `design/playout-channel-architecture-phase3.md`. This is the admin-facing surface that makes the multi-channel playout model operable; Phases 1-2 of the rethink (API + data model) shipped in the 0.2.x consolidation.
