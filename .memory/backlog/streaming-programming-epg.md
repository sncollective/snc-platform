# Programming Schedule / EPG

Time-slot-based scheduling for playout channels — `schedule` table, programming grid UI, and EPG (Electronic Program Guide) output. Enables an admin to assign playout content to time slots (e.g., "S/NC Classics plays jazz on weekday mornings, world music Friday evenings"). Deferred from Phase 5 follow-up; useful for playout programming grid once the multi-channel model is in place.
