# Stream Scheduling

A `stream_schedule` table and CRUD endpoints for creators to pre-announce upcoming live streams with a start time. Schedule entries display on the creator's channel page and feed into the programming grid. Lower priority than multi-channel model; most scheduling value comes from the EPG/programming grid once playout channels are in place.
