# VOD Publish Flow

Allow creators to publish a VOD recording to their content feed. Default visibility: `"subscribers"`. Flow: creator selects a processed recording from their recordings page, sets title/description/thumbnail, chooses visibility, and publishes — creating a `content` row linked to the recording. Scoped to Phase 7; depends on creator recordings management page and the content model unification (or at minimum a clean bridge between `playout_items` and the `content` feed).
