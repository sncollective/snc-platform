# Attestation: DaVinci Resolve Collaboration Page (Blackmagic Design)

## Paraphrased summary

The collaboration page describes DaVinci Resolve's multi-user real-time collaboration capabilities available in both Free and Studio (the page does not restrict these to Studio). Key storage claims: "DaVinci Resolve can be used with any type of storage ranging from direct attached hard drives to NAS and SAN systems." This is a direct, unqualified statement about local filesystem-compatible storage (NAS/SAN present as a local mountable path).

The page describes new "workflow integration and encoding APIs" that let developers connect asset management systems, but does not specify whether these are Free or Studio only. (The Studio product page attributes scripting APIs to Studio specifically.)

The page states compatibility with "all major file formats and post production software, making it easy to move files between DaVinci Resolve, Final Cut Pro, Media Composer, and Premiere Pro" — this implies interchange support exists but does not enumerate specific formats (EDL/AAF/FCP XML).

Blackmagic Cloud is described as enabling project and media hosting accessible globally, with "Share Cloud Folders" for remote collaboration.

## Key passages

> "DaVinci Resolve can be used with any type of storage ranging from direct attached hard drives to NAS and SAN systems."

> "New workflow integration and encoding APIs let developers integrate workflow and asset management systems with DaVinci Resolve."

> "it works with all major file formats and post production software, making it easy to move files between DaVinci Resolve, Final Cut Pro, Media Composer, and Premiere Pro"

> "The project libraries are built for real time local and remote collaboration"
