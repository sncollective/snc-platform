# Attestation: DaVinci Resolve Studio Product Page (Blackmagic Design)

## Paraphrased summary

The official Studio product page for DaVinci Resolve 21 describes Studio as a one-time perpetual purchase at USD $295 (the UK page separately confirmed £225). The page positions scripting as a Studio-exclusive capability — Python and Lua scripting are listed under the Studio feature set, with the explicit note that Studio "features support for both Python and LUA scripting, along with developer APIs that allow you to add workflow integrations and custom encoding options." The page also describes workflow integration plug-ins using JavaScript APIs for connecting to local and remote asset management and automation systems. These scripting and API features are presented as Studio differentiators from the free version; the free version comparison summary does not include them.

No mention of headless rendering, command-line rendering, or background render without GUI is present on this page.

No specific network media source protocols (HTTP URL media sources, NFS/SMB/rclone) are mentioned.
