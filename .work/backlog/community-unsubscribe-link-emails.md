# Unsubscribe Link in Emails

One-click unsubscribe per RFC 8058: add a `List-Unsubscribe` header to outgoing notification emails and provide a corresponding unsubscribe endpoint. The unsubscribe action should update the user's notification preferences without requiring authentication (token-based). Applies to all notification email types dispatched by the notification worker.
