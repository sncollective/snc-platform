# Emissions Admin Tooling

Admin UI for managing emissions entries, offsets, and integrations.

- List/filter/search entries, reusing GET `/breakdown` data
- Add entry form: manual entry or calculated — select method, enter inputs, compute co2Kg
- Add offset form
- ccusage integration: pull Claude Code token data and auto-calculate emissions
- Studio session log entry form
