# Data Portability & Export

Per-protocol and full-archive export formats — ActivityStreams JSON-LD, AT Protocol CAR files, Nostr signed JSON events, Polycentric signed event log, full JSON archive, and scheduled opt-in automated backups.

**Bullets to split / scope out when this is promoted:**

- ActivityStreams JSON-LD archive via Fedify
- CAR file — AT Protocol repository export
- Nostr event export — signed JSON events (portable to any relay)
- Polycentric signed event log
- Full JSON archive — all user data in portable format
- Scheduled backups — opt-in automated periodic exports
