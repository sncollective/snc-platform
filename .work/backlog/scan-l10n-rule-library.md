# Scan l10n Rule Library

Create a `scan-l10n` rule library under the refactor-scan family. Covers locale file completeness, missing translations, untranslated UI regions, and locale-dependent logic without fallbacks. Follow the `create-scan-*` pattern used by existing scan libraries (see [scan-design-system-rule-library.md](scan-design-system-rule-library.md) for a sibling).
