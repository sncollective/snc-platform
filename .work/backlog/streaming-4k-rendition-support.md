# 4K Rendition Support

Add a 4K transcode profile to the playout content pipeline. Requires a new rendition entry in the transcode config (3840×2160 or 2160p), storage impact assessment for Garage S3, and Vidstack quality track configuration. Deferred from Phase 5 follow-up pending clear demand and storage cost clarity.
