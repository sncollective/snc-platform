# Studio Booking Request System (Phase 2)

Authenticated booking request flow for studio sessions.

- Authenticated booking request form (extends existing service booking pattern)
- Request includes: service type, preferred dates, session details, notes
- Co-op member dashboard for incoming requests
- Status tracking (inquiry > quoted > confirmed > completed)
- Notifications on status changes
